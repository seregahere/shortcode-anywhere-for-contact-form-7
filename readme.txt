Enable shortcodes for Contact Form 7, for form and mail body.

== Description ==

This plugin enable shortcodes in forms created with Contact Form 7, and in the mail body.

Requires Contact Form 7 plugin to be installed.

== Installation ==

1. Upload the entire 'shortcode-anywhere-for-contact-form-7' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

This plugin don't have a menu, or settings.
